# QQ 农场 Bot 忘记管理员密码指南

如果您忘记了平台的前端登录管理员密码（由于采用不可逆的 PBKDF2 哈希加密，您无法直接在数据库看明文解出），可直接使用我们提供的内置修复工具一键重置。

## 操作步骤

1. 退回到项目的根目录执行重置脚本：
   ```bash
   ./reset-admin-password.sh
   ```

2. 出现提示菜单，可供您选择：
   *   按 `1`: 重置为默认密码 `admin`
   *   按 `2`: 重置为测试密码 `qq007qq008`
   *   按 `3`: 自定义输入新密码

3. 输入后按 `y` 确认操作。脚本底层将会自动为您生成正确的盐和 PBKDF2 加密串并覆盖 `core/data/users.json`，同时也会做好自动备份防丢工作。

## 获取已注册用户清单

如果你想知道系统里有哪些用户，请执行：
```bash
./show-users.sh
```
此命令将打印所有用户及其禁用状态，方便日常排查。
