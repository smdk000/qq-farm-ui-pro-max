# PLAN_业务架构解析

## 1. 目标
为用户梳理 QQ Farm Bot 2.0 升级至 2.1 版本的全景业务逻辑与架构，并利用深色主题、Mermaid 生成高级分析报告，存入所需目录。

## 2. 背景
- 系统基于 Node.js, Express, better-sqlite3 构建核心后端，Vue 3 + Pinia 构建前端面板。
- 此前进行了从 1.0 多用户版本向 2.1 极简农场自动化架构改造。
- 添加了两季作物保护、60秒极限防偷抢收、全自动好友通过等进阶逻辑。

## 3. 设计产出
- **Plan, Task, Report 文档** 位于 `docs/业务架构解析/` 内，严格遵守极简 5S 管理规范。
- **业务逻辑 HTML 分析报告** 采用自包含无外部依赖（除 Mermaid CDN）的高精美深色分析表图，直接生成于 `docs/plans/业务逻辑全景解析报告.html` 中供用户阅读查收。
