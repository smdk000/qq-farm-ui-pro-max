# 阶段 1: Align (对齐) - 侧边栏主题切换重构

## 1. 需求背景
用户希望将之前在左侧固定位置的简单主题光球切换控件，重构成类似 JeecgBoot Pro 的**右侧抽屉式（Drawer）项目配置栏**。
同时要求**重新设计**之前提供的 5 种主题，提升视觉高级感和专业度（遵循 ui-ux-pro-max 规范）。

## 2. 核心功能拆解
1. **右侧弹出抽屉组件 (ThemeSettingDrawer)**
   - 触发器：点击顶部导航栏的“设置/主题”图标（或者浮动齿轮图标）弹出。
   - 布局对标：JeecgBoot-main 中的 `SettingDrawer.tsx`。
2. **主题参数配置项**
   - **深色模式 (Dark/Light Mode)**：独立全局开关（覆盖所有）。
   - **导航栏模式 (Nav Mode)**：左侧菜单布局、顶部菜单布局（UI占位展示）。
   - **系统主题 (System Theme)**：5 个重新设计的全新高级主色调。
   - **顶栏/菜单主题 (Header/Menu Theme)**：允许顶部栏和侧边栏在亮色/暗色基调中独立切换。
   - **界面设置 (UI Settings)**：灰色模式、色弱模式、标签页显示、面包屑显示等。
3. **全新 5 套主色调设计**
   - **御农翠绿 (默认)**：`#22c55e` (经典护眼配色，代表生机与丰收)
   - **全息赛博 (Neon)**：`#8b5cf6` (电竞荧光紫，极地夜效凸显极客感)
   - **尊贵黯金 (Elegant)**：`#eab308` (高端优雅，黄黑相间质感十足)
   - **深海矩阵 (Ocean)**：`#0ea5e9` (冷静数据蓝，带来流畅的监控体验)
   - **樱花粉黛 (Sakura)**：`#ffc0cb` (活泼粉色系，纯净且轻盈)
4. **状态持久化**
   - 配置项全面接入 `Pinia` 或 `useStorage` 持久化，保证刷新浏览器后不丢失。

## 3. 技术约束与规范
- UI 框架：Vue 3.5 + Ant Design Vue 4.2.6 (现有选型) + Tailwind CSS。
- 实现方式：尽量使用 Ant Design Vue 原生的 Drawer、Divider、Switch 组件。
- 颜色操作：动态注入 CSS Variables（如 `--ant-color-primary`）来实现切换。

## 4. 验收标准
1. 输出一份 HTML Plan `Plan_20260301_侧边主题切换.html`，清晰描绘右侧抽屉的预览样式。
2. 新的 5 种主题色域需明确并在文档/计划中展示。
3. 抽屉内的设置项需符合 JeecgBoot 风格，排版整齐。
