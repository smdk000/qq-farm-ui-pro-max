🌾 农场工具插件化集成计划 (Plan)

基于非入侵式设计与跨域降级通讯架构 | 日期: 2026-03-05

🎯 目标与核心规范 (Objectives & Specs)

  • 非入侵式 (Non-invasive) 通过动态资源探测技术，在不修改原静态文件逻辑的前提下实现“即插即用”（存在即挂载，删除即隐藏）。
  • 无缝融合 (Seamless UI) Vue 侧页面风格对齐现有帮助中心，右侧以满屏展现 iframe。
  • 全主题适配 (Theme Sync) 实现跨上下文五大主题色 (默认/赛博/尊金/深海/樱花) 及深浅模式的双向通讯适配。

💥 影响面分析 (Impact Analysis)

模块 / 组件 影响类型 修改详情与应对措施

🛠️ 实施步骤任务拆解 (Implementation Steps)

  1. 目录配置阶段：配置 vite.config.ts 提供 nc_api_version 的开发环境挂载，确保 localhost:2800/nc_api_version/index.html 可访问。
  2. 探测与路由钩入阶段：
    • 在系统加载时 fetch('/nc_api_version/index.html', { method: 'HEAD' }) 判断挂载状态。
    • 动态更新导航栏菜单。
  3. 前端视图开发：新建 FarmTools.vue，实现与 HelpCenter 高度统一的左侧二级菜单界面。右侧区域移除所有 padding 与 margin （p-0 w-full），内部撑满一个隐藏 border 的 iframe。
  4. 主题桥接 (Theming Bridge) 注入：
    • 原 HTML 无需修改文件源码，前端在 iframe onload 回调中，通过 contentWindow.postMessage 推送样式指令。
    • 同时，可通过动态注入脚本（由于同源策略）直接修改 iframe.contentDocument 以应用颜色替换。对于全局背景和高亮元素（如 bg-green-400 -> bg-primary-400），注入匹配当前项目的 CSS。
  5. 自适应审查：验证 5 套主题在深色、浅色模式下对原生 iframe 内容的视觉覆盖程度。
