### 2. 搜索功能

```typescript
function handleSearch() {
  const query = searchQuery.value.toLowerCase()
  
  searchResults.value = helpArticles.filter(article => {
    if (article.title.toLowerCase().includes(query)) return true
    if (article.tags.some(tag => tag.toLowerCase().includes(query))) return true
    if (article.category.toLowerCase().includes(query)) return true
    
    return false
  }).slice(0, 10)
}
```

**特性**：
- ✅ 实时搜索
- ✅ 多字段匹配
- ✅ 结果限制
