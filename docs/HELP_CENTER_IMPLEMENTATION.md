## 📊 技术实现

### 组件结构

```
HelpCenter/
├── 顶部标题栏
│   ├── 标题 + 图标
│   └── 搜索框
├── 左侧分类导航
│   ├── 分类标题
│   ├── 分类图标
│   └── 文章列表
└── 主内容区
    ├── 文章标题
    └── 文章内容
        ├── 信息卡片
        ├── 提示框
        └── 代码块
```

### 状态管理

```typescript
interface HelpState {
  categories: Category[];      // 分类列表
  selectedCategory: string;    // 当前分类
  selectedArticle: string;     // 当前文章
  searchQuery: string;         // 搜索关键词
}
```

### 路由配置

```typescript
{
  path: 'help',
  name: 'help',
  label: '帮助中心',
  icon: 'i-carbon-book',
  component: () => import('@/views/HelpCenter.vue'),
}
```

## 🎯 使用指南

### 访问帮助中心

**方法一：侧边栏菜单**
1. 查看左侧菜单底部
2. 点击"📚 帮助中心"
3. 进入帮助中心页面

**方法二：悬浮按钮**
1. 点击右下角悬浮❓按钮
2. 自动跳转到帮助中心

### 查找帮助内容

**浏览方式**：
1. 左侧选择分类
2. 展开分类列表
3. 点击具体文章

**搜索方式**：
1. 顶部搜索框输入关键词
2. 按回车或点击搜索按钮
3. 查看搜索结果

### 内容结构

**5 大分类**：
1. 📖 新手入门 - 快速上手指南
2. ⚙️ 设置指南 - 功能详细说明
3. 🎯 高级功能 - 进阶玩法
4. 🔧 故障排查 - 常见问题解决
5. 📋 配置模板 - 直接套用的配置
