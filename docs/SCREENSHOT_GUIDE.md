## 🎯 准备工作

### 环境配置

1. **浏览器设置**
   ```
   推荐浏览器：Chrome 120+ / Edge 120+
   分辨率：1920x1080 或 1600x900
   缩放：100%（不要使用浏览器缩放）
   主题：深色模式（推荐）或浅色模式
   ```

2. **开发者工具设置**
   ```
   F12 打开开发者工具
   切换到设备工具栏（Ctrl+Shift+M）
   选择"Responsive"或固定分辨率
   禁用缓存（Network 标签）
   ```

3. **账号准备**
   ```
   - 管理员账号：admin / admin（或自定义）
   - 测试账号：至少 1-2 个已登录的农场账号
   - 数据准备：确保有作物、好友、日志等数据
   ```

### 截图工具推荐

**Windows:**
- Snipaste（推荐）- 免费、功能强大
- ShareX - 开源、支持滚动截图
- Windows 自带截图（Win+Shift+S）

**macOS:**
- 系统自带截图（Cmd+Shift+4/5）
- CleanShot X（付费）
- Snipaste（跨平台）

**Linux:**
- Flameshot（推荐）
- Shutter
- System screenshot tool

## ✨ 最佳实践

### Do's（推荐）

✅ **使用深色模式** - 更现代、更专业  
✅ **保持数据真实** - 使用实际的农场数据  
✅ **统一风格** - 所有截图使用相同主题和分辨率  
✅ **敏感信息打码** - 密码、个人信息等  
✅ **展示活跃状态** - 账号在线、有日志滚动  
✅ **包含上下文** - 显示侧边栏、导航等  

### Don'ts（避免）

❌ **使用模糊截图** - 文字必须清晰可读  
❌ **展示敏感数据** - 真实密码、Cookie 等  
❌ **空数据页面** - "暂无数据"很难看  
❌ **混合主题** - 不要有些深色有些浅色  
❌ **过小的窗口** - 不要小于 1280x720  
❌ **杂乱的背景** - 关闭无关的浏览器标签

## 🔧 常见问题

### Q1: 截图有摩尔纹怎么办？
**A:** 不要用手机拍屏幕！使用系统截图工具或 Snipaste。

### Q2: 地址栏要不要显示？
**A:** 推荐显示，可以展示 `localhost:3000` 增加真实感。但确保不暴露真实 IP。

### Q3: 如何展示深色模式？
**A:** 
1. 点击右上角主题切换按钮
2. 切换到深色模式
3. 等待动画完成
4. 立即截图

### Q4: 日志区域总是空的怎么办？
**A:** 
1. 启动一个账号
2. 等待自动操作产生日志
3. 或手动触发一些操作（如收获）
4. 快速截图

### Q5: 如何统一所有截图的色调？
**A:** 
1. 固定浏览器窗口大小
2. 使用相同的缩放比例
3. 在同一时间段截图（光线一致）
4. 后期用工具批量调整
