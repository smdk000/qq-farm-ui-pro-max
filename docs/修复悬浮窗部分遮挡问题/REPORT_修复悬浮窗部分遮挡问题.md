# 修复悬浮窗部分遮挡问题 - 总结报告

## 修复实施结果
根据最新规划与 UI 改造动作，我们对悬浮内容遮挡问题进行了以下深度优化与重构：
1. **清理多余代码**：在 `BaseSwitch.vue` 组件中，因悬浮提示内容已整合至开关按钮下方直接展示，我们彻底移除了已废弃的浮窗弹出代码（包含图文触发器及 `.hint-bubble` 相关 CSS 样式），从根源上消灭了开关自带气泡遮挡的可能。
2. **样式重构与全局防丢解法**：在 `web/src/style.css` 中排查修复了引起类似弹窗与下拉框遮挡的底层原因。修改了 `.glass-panel` 的 `contain: layout style paint;` 属性，移除了导致边距裁剪阻断的 `paint` 包含作用域，从而保证未来诸如 `BaseSelect.vue`（例如：施肥策略下拉框）等需要绝对定位跨越父容器边界的元素不再产生截断问题。

## 验收说明
- 农田设置和控制面板各选项下，原有的悬浮（Tooltip）不再触发（已精简），所有提示文案通过文字描述直观呈现。
- "施肥策略"等下拉框由于解除了父容器 `paint` 剪裁控制，不再出现下拉悬浮显示不全或断层遮挡的情况。
