# 任务拆解：优化 Admin 界面布局

## 1. 任务列表
1. **[ ] 优化 `UserInfoCard.vue` 布局**
   - 移除原有占据多行的网格布局。
   - 将“卡密类型”、“到期时间”、“剩余时间”移动至顶部信息栏，与用户名水平或紧凑排列。
   - 将“续费”、“改密”按钮变为小尺寸的幽灵按钮或圆角按钮，与“退出”按钮并列排放在右侧。
   - 如果有到期预警提示，将其融合到组件的下边缘或紧密结合在信息区域。

2. **[ ] 优化 `Dashboard.vue` 的账号与统计卡片**
   - 缩小间距：将 3 个顶级状态卡片的 padding 调小。
   - 账号经验卡片 (`Account & Exp`)：减少上下外边距，优化经验条显示空间。
   - 资产卡片 (`Assets & Status`)：简化边框/线条的上下间距 (`mt-4`, `pt-3` 改为 `mt-2`, `pt-2`)。
   - 物品卡片 (`Items`)：优化化肥和收藏点部分的间隙。

3. **[ ] 更好的显示“今日统计”**
   - 由于上面节约了大量的垂直高度，左侧日志和右侧的“今日统计”将自然获得更多高度。
   - “今日统计”本身的卡片高度限制（`min-h-0`, `flex-1`）在弹性容器下将使得它扩展，我们可以移除或调优右列的 `flex` 权重，确保其不会被压扁。

## 2. 依赖项与执行顺序
按任务列表顺序执行。均可以独立编译热更新并在浏览器验证。
