# 农场工具集成任务 (Align & Consensus)

## 任务目标
将位于项目根目录的 `nc_api_version` 文件夹作为一个辅助查询工具“插件”，非入侵式地接入到前端控制台中，暴露入口在帮助中心旁或侧边栏下方。

## 关键需求
1. **插件式设计**：系统必须动态检测 `nc_api_version/index.html` 是否存在，存在则挂载此功能菜单，被删除则该功能彻底隐藏。
2. **UI 高度统一**：界面需要跟“帮助中心”的风格一致（左边拥有分类二级菜单，右边为内容区域）。重点要求是**右侧区域屏铺**，最大化利用版面。
3. **跨上下文支持**：无缝兼容系统的原生5款主题（御农翠绿、全息赛博、尊贵黯金、深海矩阵、樱花粉黛）以及深浅模式。

## 技术约束与疑点
1. **静态资源挂载**：`nc_api_version` 目前并不是在 Vite 的 `public` 目录下，而是在项目根目录。要在开发环境和生产环境通过 HTTP 被读取，需要设置 Vite 服务器资源代理及构建后 Nginx/Docker 容器的 Volume 映射规则。
2. **同源策略与主题注入**：鉴于 `nc_api_version` 下有 5 个静态 HTML，其本身硬编码了 Tailwind 的 `farm` 与 `green` 色系，不支持暗黑模式且无法感知外部 Vue 应用的主题。由于将同源部署（同一端口），可以在父组件（Vue）加载 `iframe` 时，直接操作 `iframe.contentDocument` 动态向子 HTML `<head>` 注入 `<style>`，强制覆写相关 Tailwind 原生类来实现。

## 验收标准
- [ ] 删除根目录 `nc_api_version` 刷新后，左侧菜单隐藏。加入该文件夹，自动重现。
- [ ] 进入农场工具后，左侧拥有统一拟态玻璃效果的二级菜单，右侧以满屏显示原站内容。
- [ ] 切换任何系统主题与深浅模式，iframe 内部内容实时无缝跟随更新（页面不白边，不突兀）。。
