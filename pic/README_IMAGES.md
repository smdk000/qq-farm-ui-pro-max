## 图片格式要求

- **格式**: PNG（推荐）或 JPG
- **质量**: 清晰可辨，文字不模糊
- **文件大小**: 建议每张图片 < 500KB
- **命名**: 使用小写字母和连字符，如 `dashboard.png`

## 截图建议

1. **使用深色模式截图** - 更符合现代审美
2. **展示完整界面** - 包含侧边栏和主要内容区
3. **敏感信息打码** - 如账号密码、个人信息等
4. **统一浏览器窗口大小** - 建议 1920x1080 或 1600x900
5. **保持风格一致** - 所有截图使用相同的主题和窗口大小

## 临时方案
