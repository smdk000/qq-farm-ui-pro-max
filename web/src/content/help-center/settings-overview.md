# 设置页总览

当前版本的设置页已经从单页堆叠，升级成按场景组织的多分类工作台。

## 六个主分类

### 常用设置

- 适合第一次配置或日常快速改几个关键项。
- 聚合当前账号策略、密码、通知和外观相关的高频入口。

### 种植策略

- 关注选种、库存规则、巡查区间和收获节奏。

### 自动任务

- 集中管理农场动作、好友互动、施肥和任务领取。

### 通知提醒

- 管理下线提醒、经营汇报、历史记录和渠道文案。

### 账号与安全

- 管理账号模式、风控提示、高风险窗口和敏感动作。
- 管理员还可以直接看到“卡密发放总控”，统一控制注册是否必须卡密，以及续费、体验卡与后台发码能力。

### 高级设置

- 仅管理员可见。
- 包含系统体检、时间参数、系统更新、集群流控、主题外观、体验卡配置和第三方 API。

## 阅读方式建议

### 普通用户

优先顺序建议是：

1. 常用设置
2. 自动任务
3. 账号与安全
4. 通知提醒
5. 种植策略

### 管理员

在普通用户顺序之外，还要补看“高级设置”。

## 当前版本的重点变化

### 页面不是简单折叠

每个主分类都有自己的引导说明、快捷配置和详细配置区，不建议再按旧版“从上翻到下”的方式使用。

### 风险提示更前置

QQ 高风险项、账号模式、静默时段和提醒配置都被前置了，目的就是减少误开。

### 高级能力单独分区

主题、更新、API、集群和体检信息都放到了管理员高级区，避免普通用户误触。
